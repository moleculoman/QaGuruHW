TC_1: Проверка на то, что строка поиска работает и возвращает результаты
1. Открыть страницу https://megamarket.ru/
2. Нажать на кнопку "Найти товары"
3. Ввести значение "vr" в строке поиска
4. Нажать кнопку "Искать"
Ожидаемый результат:
В результатах поиска найден 1 или более товаров по тематике "Очки виртуальной реальности".


TC_2: Проверка наличия определенной категории VR
1. Открыть страницу https://megamarket.ru/
2. Нажать на кнопку "Найти товары"
3. Ввести значение "vr для телефона" в строке поиска
4. Нажать кнопку "Искать"
5. Перейти в раздел "Категория"
Ожидаемый результат:
Среди разделов имеется раздел с наименованием "Очки виртуальной реальности для смартфона"


TC_3: Проверка поиска определенной модели VR-шлемов
1. Открыть страницу https://megamarket.ru/
2. Нажать на кнопку "Найти товары"
3. Ввести значение "quest 3" в строке поиска
4. Нажать кнопку "Искать"

Ожидаемый результат:
Среди результатов поиска выбран раздел с наименованием "Очки виртуальной реальности" и среди товаров, имеется товар
с наименованием вида : Значение из п.3 ТК + слово "шлем"
